## Descripción del proyecto:
El proyecto consiste en la creación de una aplicación para
un cine que gestiones reservas y compras de entradas y 
alimentos. La aplicación debe permitir a los usuarios 
egistrarse, loguearse, ver la cartelera, reservar y comprar entradas y alimentos, 
ver sus compras y reservas, y ver información sobre las películas.

Para iniciar la aplicación, se debe ingresar al siguiente link:

Una vez ingresado, se verá en pantalla la homepage. Para logearse o crear un nuevo usuario se derberá clickear el botón de arriba a la 
derecha. Una vez logeado, se podrá ver la cartelera y seleccionar las películas que se desean ver y/o alimentos a comprar.

Estando logueado, también se pueden acceder a las compar activas de dicho usuario, tanto de snacks como de entradas para películas.

## Configuración general del proyecto
El archivo application.properties contiene la configuración de la base de datos.  Las credenciales para poder acceder a la base de datos están en el archivo .env.
